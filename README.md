# Poopal Project

Hello World! 🐕💩

欢迎来到Poopal智能物联网宠物训练系统！

## 项目简介

Poopal是一个基于人工智能和物联网技术的革命性宠物训练解决方案。本项目专注于解决城市宠物主最头疼的问题之一：如何训练小狗在指定位置排便。

### 核心功能
- **智能检测系统**：采用先进的计算机视觉算法和传感器技术，实时监测小狗的排便行为和位置
- **精准定位识别**：通过AI视觉识别和传感器融合，准确判断排便是否发生在指定的尿垫区域
- **即时奖励机制**：当检测到正确行为时，系统自动触发奖励装置（零食投放、声音提示等）
- **行为数据分析**：记录和分析宠物的排便习惯，为主人提供训练进度报告和建议
- **移动端控制**：通过手机App远程监控、设置参数和查看训练数据

## 🎁 奖励系统详解

### 多元化奖励机制
Poopal采用多感官奖励体系，确保宠物能够快速建立正确的条件反射：

#### 1. 自动零食投放系统
- **精密投放器**：采用螺旋式投放机构，确保每次投放1-3颗零食
- **零食识别**：支持多种尺寸的训练零食，自动调节投放量
- **防卡死设计**：智能检测投放状态，避免零食卡住
- **容量监控**：实时监测零食剩余量，通过App提醒补充

#### 2. 声音奖励系统
- **音效库**：内置多种奖励音效（铃声、赞美声、主人录音等）
- **音量调节**：根据环境噪音自动调节音量，确保宠物能听到
- **个性化定制**：支持上传主人声音，增强奖励效果
- **时间控制**：避免夜间扰民，支持静音时段设置

#### 3. 视觉反馈系统
- **LED指示灯**：绿色表示正确行为，红色表示错误行为
- **呼吸灯效果**：柔和的灯光变化，不会惊吓宠物
- **亮度自适应**：根据环境光线自动调节LED亮度

#### 4. 触觉奖励（可选模块）
- **震动提醒**：轻微震动提醒宠物注意
- **温度反馈**：加热垫提供温暖舒适感

## 📡 智能视觉检测系统

### AI计算机视觉核心

#### 1. 高清摄像头系统
- **摄像头规格**：1080P高清分辨率，120°广角镜头覆盖2m×2m区域
- **夜视功能**：红外补光灯，实现24小时全天候监控
- **图像优化**：自动曝光、自动白平衡、智能降噪处理
- **防护设计**：IP65防水防尘，耐咬材质，适应宠物环境

#### 2. 深度学习视觉算法
- **宠物目标检测**：基于YOLOv8的实时宠物检测模型，精确识别小狗
- **行为姿态分析**：深度学习姿态估计，准确识别蹲下、排便、离开等关键动作
- **区域定位判断**：计算机视觉算法精确判断宠物位置是否在指定尿垫区域
- **时序行为分析**：连续视频帧分析，确保行为识别的连续性和准确性

#### 3. 智能人员识别与过滤
- **人体检测算法**：AI自动识别人员进入监控区域
- **智能忽略机制**：检测到人员时系统继续工作，但仅记录不触发奖励
- **多目标跟踪**：同时跟踪和区分人员与宠物，精确定位训练目标
- **环境适应性**：适应家庭复杂环境，准确区分不同对象

#### 4. PIR运动检测辅助
- **智能唤醒**：PIR传感器检测到运动时激活摄像头，节能高效
- **预警机制**：提前0.5秒激活视觉系统，确保不错过关键时刻
- **误触过滤**：结合AI视觉识别，自动过滤非目标移动
- **检测范围调节**：可调节检测灵敏度，适应不同空间大小

#### 5. 重量传感器验证
- **精确称重**：四角高精度称重传感器，检测±10g重量变化
- **排便确认**：结合视觉识别，双重验证排便行为的真实性
- **异常监测**：检测异常重量变化，及时发现健康问题
- **多点校准**：四点称重确保检测精度和稳定性

### 核心技术特色
- **边缘AI计算**：所有AI推理在本地进行，保证毫秒级响应和隐私安全
- **自适应学习**：系统持续学习每只宠物的独特行为模式，个性化优化识别准确性
- **多模态数据融合**：视觉、运动、重量传感器数据智能融合，识别准确率达95%+
- **智能抗干扰**：AI算法自动识别并忽略人员活动、玩具移动等干扰因素

## 🏠 应用场景分析

### 1. 城市公寓家庭
**场景特点**：空间紧凑、邻里关系重要、清洁要求高、家庭成员活动频繁
- **解决方案**：
  - 静音模式设计，避免扰民
  - 紧凑型设备，节省空间
  - 高效清洁提醒，维护卫生
  - AI智能识别家庭成员，正常训练不受干扰
- **适用宠物**：小型犬、中型犬
- **效果预期**：2-4周建立稳定排便习惯

### 2. 独立住宅
**场景特点**：空间充足、可设置专门训练区域、访客较多
- **解决方案**：
  - 大范围检测覆盖，多区域设置
  - 户外训练模式支持
  - 全天候监控功能
  - 智能识别访客，自动调整训练模式
- **适用宠物**：各种尺寸宠物
- **效果预期**：1-3周完成基础训练

### 3. 办公室宠物友好环境
**场景特点**：白天长时间独处、员工活动频繁、需要安静
- **解决方案**：
  - 远程监控功能，主人实时了解训练状态
  - 自动化程度高，无需人工干预
  - 静音工作模式，不影响办公环境
  - AI识别员工活动，训练照常进行但避免误触发
- **适用宠物**：温顺小型犬
- **效果预期**：配合日常训练，4-6周见效

### 4. 宠物训练机构
**场景特点**：专业训练、多宠物管理、训练师频繁活动
- **解决方案**：
  - 多设备联网管理，统一监控
  - 详细数据记录，专业分析报告
  - 个性化训练方案，因狗制宜
  - AI识别训练师，不影响专业训练流程
- **适用宠物**：各类训练犬
- **效果预期**：1-2周快速建立条件反射

### 5. 特殊需求场景
#### 老年犬重新训练
- 延长奖励间隔，适应老年犬反应速度
- 增强视觉识别敏感度，补偿感知能力下降
- AI算法适配老年犬行为特征

#### 多宠物家庭
- 个体识别功能，区分不同宠物并分别训练
- 防止宠物间干扰，独立训练空间管理
- 多目标跟踪，确保训练精准度

#### 康复期宠物
- 配合医疗建议，调整训练强度和模式
- 行为异常检测，及时发现健康问题
- 温和训练模式，避免应激反应
