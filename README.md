# Poopal Project

Hello World! 🐕💩

欢迎来到Poopal智能物联网宠物训练系统！

## 项目简介

Poopal是一个基于人工智能和物联网技术的革命性宠物训练解决方案。本项目专注于解决城市宠物主最头疼的问题之一：如何训练小狗在指定位置排便。

### 核心功能
- **智能检测系统**：采用先进的传感器技术和计算机视觉算法，实时监测小狗的排便行为和位置
- **精准定位识别**：通过多重传感器融合，准确判断排便是否发生在指定的尿垫区域
- **即时奖励机制**：当检测到正确行为时，系统自动触发奖励装置（零食投放、声音提示等）
- **行为数据分析**：记录和分析宠物的排便习惯，为主人提供训练进度报告和建议
- **移动端控制**：通过手机App远程监控、设置参数和查看训练数据

## 🎁 奖励系统详解

### 多元化奖励机制
Poopal采用多感官奖励体系，确保宠物能够快速建立正确的条件反射：

#### 1. 自动零食投放系统
- **精密投放器**：采用螺旋式投放机构，确保每次投放1-3颗零食
- **零食识别**：支持多种尺寸的训练零食，自动调节投放量
- **防卡死设计**：智能检测投放状态，避免零食卡住
- **容量监控**：实时监测零食剩余量，通过App提醒补充

#### 2. 声音奖励系统
- **音效库**：内置多种奖励音效（铃声、赞美声、主人录音等）
- **音量调节**：根据环境噪音自动调节音量，确保宠物能听到
- **个性化定制**：支持上传主人声音，增强奖励效果
- **时间控制**：避免夜间扰民，支持静音时段设置

#### 3. 视觉反馈系统
- **LED指示灯**：绿色表示正确行为，红色表示错误行为
- **呼吸灯效果**：柔和的灯光变化，不会惊吓宠物
- **亮度自适应**：根据环境光线自动调节LED亮度

#### 4. 触觉奖励（可选模块）
- **震动提醒**：轻微震动提醒宠物注意
- **温度反馈**：加热垫提供温暖舒适感

## 📡 红外线检测系统

### 多层次检测架构

#### 1. PIR人体感应模块
- **高精度传感器**：检测宠物移动和停留
- **范围设置**：可调节检测范围，适应不同尺寸空间
- **误触过滤**：智能算法过滤非目标移动（如人员走动）

#### 2. 红外线栅栏阵列
- **精确定位**：8x8红外线矩阵，精确定位宠物位置
- **高度检测**：多层红外线设置，检测蹲下、站立等姿势
- **实时监控**：毫秒级响应，确保行为检测的及时性

#### 3. 热成像辅助系统
- **温度检测**：识别宠物体温特征，区分宠物和其他物体
- **行为分析**：通过热量分布分析宠物行为状态
- **环境补偿**：根据环境温度自动校准检测阈值

#### 4. 光束中断检测
- **边界监控**：监测宠物是否进入指定区域
- **方向判断**：识别宠物进入和离开的方向
- **多点布防**：关键位置设置多个检测点，提高准确性

### 智能融合算法
- **多传感器数据融合**：结合所有传感器数据，提高检测准确性
- **机器学习优化**：根据宠物行为特征不断优化检测算法
- **误报消除**：智能识别并过滤虚假触发信号
